Projet Personnel Encadré_

La réalisation d'un projet PPE est essentielle afin de valider notre BTS.
Le thême de ce dernier est le Contexte GSB.

Présentation du Projet
Le projet qui nous a été proposé durant notre BTS SIO est une Application Web gérant les notes de frais des employés d'une entreprise lambda. Cette application comprend différentes pages accessibles par les employés , ou les comptables , en fonction des pages. Les employés rentreront leurs fiches de frais sur cette application ; par la suite , ces fiches seront classées et validées ou non par les comptables.

Outils utilisés
La réalisation de ce projet a nécessité l'utilisation de différents software:
¤ HTML5 et CSS3 pour la création des pages web
¤ Wamp , afin d'utiliser PHPMyAdmin pour notre base de données
¤ Symfony2, pour lier notre base de données à nos page web, mais aussi pour faciliter la création de l'application.

Répartition du Projet
L'aboutissement de ce projet a nécessité plusieurs semaines de travail , c'est pourquoi nous avons du partager le travail à effectuer entre tous les membres de notre groupe. Nous avons jugé nécessaire de séparer, dans un premier temps, la partie comptable et la partie visiteur. Par la suite , le début du travail a consisté en la création de la base de données , ainsi que les maquettes des pages web. A l'aide des langages vus en cours , nous avons pu réaliser les différentes pages twig de l'application , ainsi que la base de données. L'utilisation de Symfony2 étant demandée pour ce projet, nous avons du regarder la documentation ainsi que de nombreux tutoriels afin de trouver comment développer notre application sur celui-ci, parmi eux:
- La création du FOSUser Bundle, qui va permettre de gérer la page de connexion, et d'envoyer le visiteur ou le comptable vers les bonnes pages
- la création des formulaires
- la création des controllers
- la création des routes
Il est possible d'accéder a l'application en suivant ce lien.

Deuxième Projet_

Le deuxième projet que nous avons réalisé est une Application Mobile permettant aux employés de consulter leurs fiches de frais ; c'est une application liée à la première. Nous avons dû créer des requêtes renvoyant les informations de la base de données sur un service web. Par la suite, nous avons récuperé ces requêtes sur Android Studio , afin de les réutiliser pour créer notre Application Mobile.

Outils utilisés
La réalisation de ce projet a nécessité l'utilisation de différents software:
¤ Wamp , afin d'utiliser MySQL pour notre base de données
¤ Symfony2, pour paramètrer notre Application Web et envoyer des requêtes au service Web
¤ Android Studio pour le développement de l'application Android et pour la récupération des données du service Web
